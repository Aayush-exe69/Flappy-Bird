# Flappy Bird 

Hi! This is my own version of the classic **Flappy Bird** game, made using the as a beginner project.

This game was created to practice my game development skills and learn more about how 2D games work in Godot. It may look simple, but I learned a lot while building it.



---

## About the Game
The goal is simple:
Control the bird, avoid the pipes, and try to get the highest score possible.

The longer you survive, the harder it gets!

---

## Features
- Classic Flappy Bird gameplay
- Simple and clean UI
- Score tracking
- Restart system
- Main menu
- Endless gameplay

---

## Built With
- GDScript

---

## Controls
- **Spacebar** or **Mouse Click** to make the bird flap
- Avoid obstacles and survive as long as you can

---

## Why I Made This
I built this game as part of my learning journey in game development.  
It helped me understand:
- Signals
- Collision detection
- Scene management
- UI design
- Basic game logic

As a beginner, this project taught me a lot and gave me confidence to build more games.
